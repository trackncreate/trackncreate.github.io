Description

Password protected diary App in MIT App Inventor

Overview

The Diary App is a beginner-friendly project built in MIT App Inventor, Users can enter a password, which are validated before accessing a personalized diary, categorized based on date. This project is perfect for learning basic UI design, logic blocks, and data validation in MIT App Inventor, while gaining practical experience in mobile app development.

How to Use

  1. No setup needed just open the project in MIT App Inventor, for further development or exporting.
