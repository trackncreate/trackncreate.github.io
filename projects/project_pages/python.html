Description

Python editor and online interpreter App in MIT App Inventor

Overview

The python editor App is a beginner-friendly project built in MIT App Inventor,it uses an online interpreter website as an iframe to run Python. Users can type in python code and execute the code through the interpreter. This project is perfect for learning basic UI design, logic blocks, web and file management in MIT App Inventor, while gaining practical experience in mobile app development.

How to Use

  1. No setup needed just open the project in MIT App Inventor, for further development or exporting.
  2. Warning: 1)the project hasn't tested recently so the code might break   2)The interpreter website is not creative common, don't use for commercial purposes (only for education purpose).
